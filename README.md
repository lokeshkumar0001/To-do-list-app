# To-do-list-app
A simple to do list app.Built using nodejs, express, mongodb, ejs.

Funtions:

1. Add your activity
2. Remove if you have completed.

What if you want to go personal to-do ?

It's very simple just add your personal to do name after base url. eg: `{base_url}/{personal to-do name}` <br />
```https://to-do-list-app-zupq.onrender.com/{personal to-do name}```
